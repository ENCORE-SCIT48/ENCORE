# ENCORE 테마 색상 가이드

**컨셉**: 무대를 잇다 — 공연·공연장·공연자 매칭 플랫폼  
**현재 방향**: 레드(무대 커튼) + 골드(스포트라이트/프리미엄) + 따뜻한 중성색

---

## 1. 현재 테마 색상 정리

| 변수 | 현재 값 | 역할 | 느낌 |
|------|---------|------|------|
| **--encore-primary** | `#B91C2E` | 메인 액션, 강조, 링크 | 무대 커튼 레드(차가운 레드) |
| **--encore-primary-hover** | `#9B2335` | 버튼/링크 호버 | 더 진한 레드 |
| **--encore-primary-light** | `rgba(185,28,46,0.12)` | 배경 강조, 포커스 링 | 은은한 레드 틴트 |
| **--encore-primary-glow** | `rgba(185,28,46,0.25)` | 버튼 그림자, 카드 호버 | 부드러운 레드 글로우 |
| **--encore-gold** | `#B8860B` | 푸터 라인, 배지, 포인트 | 다크 골드(황금빛) |
| **--encore-gold-light** | `#D4A84B` | 푸터 링크 호버 등 | 밝은 골드 |
| **--encore-gold-muted** | `rgba(184,134,11,0.2)` | 탑바 하단 라인 | 연한 골드 |
| **--encore-bg** | `#FAF9F7` | 페이지 배경 | 따뜻한 크림 화이트 |
| **--encore-text** | `#1a1a1a` | 본문 텍스트 | 거의 검정 |
| **--encore-text-muted** | `#5c5c5c` | 보조 텍스트 | 중간 회색 |
| **--encore-footer-bg** | `#1a1a1a` | 푸터 배경 | 다크 |

- **primary**: 파란 기가 살짝 도는 레드(crimson 계열) → 무겁고 클래식한 무대 느낌.
- **gold**: 다크 골드 → 레드와 잘 어울리지만, 대비가 크지 않아서 은은한 포인트용으로 적합.
- **bg**: 약간 노릇한 흰색 → 레드/골드와 충돌 없이 읽기 편함.

---

## 2. 어울리는지 / 바꿀지 관점

### 잘 맞는 점
- **레드 + 골드**는 오페라·극장 브랜딩에서 전통적으로 쓰이는 조합이라 “무대를 잇다”와 잘 맞음.
- **따뜻한 배경(#FAF9F7)** 때문에 레드가 너무 튀지 않고, 골드는 악센트로만 쓰이기 좋음.
- **다크 푸터**로 하단이 무겁게 잡혀 있어서 전체 톤이 정돈됨.

### 개선 여지
- **primary**가 조금 어둡고 차갑게 느껴질 수 있음 → “친근한 공연 플랫폼”을 강조하려면 **조금 더 따뜻한 레드**로 바꿀 수 있음.
- **골드**를 버튼/CTA에 더 쓰고 싶다면, 현재 골드는 어두워서 **밝은 골드/앰버**를 보조 포인트로 추가하는 것도 방법.
- **대비**: primary(`#B91C2E`)를 작은 본문에만 쓰면 가독성이 떨어질 수 있으니, **강조·버튼·큰 텍스트**에 주로 쓰는 현재 용도가 적절함.

## 3. 대안 팔레트 (바꿀 때 참고)

### A. 현재 유지 + 미세 조정
- **primary만 살짝 따뜻하게**  
  - 현재: `#B91C2E`  
  - 제안: `#C41E3A`(카디널 레드), `#BE2A3C`(따뜻한 크림슨)  
- **gold-light를 조금 더 스포트라이트 느낌으로**  
  - 현재: `#D4A84B`  
  - 제안: `#E5B84B`, `#D4A574`(앰버 골드)

### B. 조금 더 “밝고 친근한” 무대
- **primary**: `#DC2626`(Tailwind red-600) — 더 밝고 따뜻한 레드.  
- **primary-hover**: `#B91C1C`(red-700).  
- **gold**: `#CA8A04`(amber-600), **gold-light**: `#EAB308`(amber-500) — 무대 조명 느낌.  
- **bg**: 현재 `#FAF9F7` 유지 또는 `#FEFCE8`(연한 옐로우 화이트)로 더 따뜻하게.

### C. “고급·다크” 극장 느낌
- **primary**: 현재 `#B91C2E` 유지.  
- **gold**: `#D4AF37`(클래식 골드), **gold-light**: `#F4E4BC`.  
- **bg**: `#F5F3EF`(약간 더 그레이 크림).  
- **footer**: `#0d0d0d`로 더 깊게.

---

## 4. 추천

- **지금 톤이 마음에 들면**  
  → **A안**처럼 primary/gold만 1~2단계 미세 조정해도 충분함.
- **“더 밝고 접근하기 쉬운” 이미지**를 원하면  
  → **B안**으로 primary를 따뜻한 레드로, 골드는 앰버 계열로 바꾸는 걸 권장.
- **“클래식·프리미엄 극장”**을 더 강조하려면  
  → **C안**으로 골드와 다크 푸터만 강화.

## 5. theme.css에서 바꿀 때

`src/main/resources/static/css/theme.css` 상단 `:root` 블록에서 아래만 교체하면 됨.

```css
/* 예: B안 적용 시 */
--encore-primary: #DC2626;
--encore-primary-hover: #B91C1C;
--encore-primary-light: rgba(220, 38, 38, 0.12);
--encore-primary-glow: rgba(220, 38, 38, 0.25);
--encore-gold: #CA8A04;
--encore-gold-light: #EAB308;
--encore-gold-muted: rgba(202, 138, 4, 0.2);
```

원하면 **A/B/C 중 하나를 골라서** 그 값으로 `theme.css` 수정까지 이어서 적용해 줄 수 있음.
