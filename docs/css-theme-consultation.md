# CSS · 테마 상담 (PC 메인, 발표용, ENCORE 주제)

발표 시연이 큰 화면이라 **PC 메인**으로 가면서, **깔끔·트렌디·무대를 잇다 ENCORE**가 잘 드러나도록 정리한 상담 노트입니다.

---

## 1. PC / 모바일 이원화 (발표용 가독성)

### 현재
- 일부 페이지가 `max-width: 420px ~ 430px` 로 모바일 중심.
- 폰트는 Bootstrap/Litera 기본(16px) 수준.

### 제안: PC 메인 + 발표용 크기

| 구분 | 제안 | 비고 |
|------|------|------|
| **레이아웃** | PC 기본 `max-width: 960px` 또는 `1120px`, 모바일은 `@media (max-width: 768px)` 에서 `max-width: 100%` 등으로 전환 | 큰 화면에서도 한 줄이 너무 길어지지 않게 |
| **폰트** | **기본보다 살짝 크게** — `html { font-size: 18px; }` (또는 17px). 모바일에서만 `17px` / `16px` 로 축소 | 발표 시 뒤에서도 읽기 좋게 |
| **제목/강조** | `h1` 1.75rem~2rem, 카드 제목 등도 1.1~1.2rem 수준으로 여유 있게 | 올드해 보이지 않게, 가독성만 확보 |

**적용 예시 (공통 theme에서):**
```css
/* PC 메인 (발표용 가독성) */
html { font-size: 18px; }
body { font-size: 1rem; color: var(--encore-text, #333333); }

@media (max-width: 768px) {
  html { font-size: 16px; }
}
```

페이지별로 이미 `max-width: 420px` 를 쓰는 경우, **PC에서는 이 제한을 풀고** 768px 이하에서만 420px 를 적용하는 식으로 바꾸면 됩니다.

## 2. 현재 테마 정리

| 구분 | 내용 | 비고 |
|------|------|------|
| Base | Bootswatch **Litera** (일부 페이지) / Bootstrap 기본 (일부) | 혼용 중 |
| Main | **#E63946** (Red) — 버튼, 로고, 강조 | 산발적으로 인라인/파일별 사용 |
| Sub | **#333333** (Dark Grey) — 텍스트, 푸터, 헤더 | 동일 |

**느낌:** Litera는 깔끔하고 가독성 좋지만, 밝은 레드(#E63946) + 회색만 쓰면 “무대/ENCORE”보다는 “일반 서비스”에 가깝게 보일 수 있음.

---

## 3. “무대를 잇다 ENCORE” 에 맞추는 방향

- **깔끔한데 올드하지 않게** → 여백 충분히, 서체 크기 적당히, 그림자/라운드는 절제.
- **트렌디한데 과하지 않게** → 포인트 컬러 1~2개 + 흰색/오프화이트 + 다크 그레이.
- **무대/ENCORE** → “다시 부른다”, “커튼콜”, “스포트라이트” 연상이 살짝 나오면 좋음.

### 옵션 A: 색상 유지 + 톤만 조정 (가장 부담 적음)

- **#E63946** 그대로 두고, **배경**을 순백(#FFF) → **약한 웜 화이트** (#FAF9F7, #F8F6F3) 로만 바꾸기.
- 푸터/헤더 **#333333** 유지하거나, **#2C2C2C** 정도로 살짝 더 묵직하게.
- 효과: Litera 느낌 유지하면서, 조금 더 “공간/무대” 같은 따뜻함만 추가.

### 옵션 B: 메인 컬러를 “무대 레드”로만 살짝 변경

- **현재 #E63946** (밝은 레드) → **딥 레드** 로만 교체  
  예: **#B91C2E**, **#9B2335**, **#A0152E**  
- 서브 #333333 유지.
- 효과: 레드가 “포스터/커튼”에 가깝게 보이고, 과하지 않게 정돈된 인상.

### 옵션 C: 포인트에 골드/앰버 추가 (ENCORE 강조)

- **메인**: #E63946 또는 옵션 B의 딥 레드.
- **서브**: #333333.
- **세컨드 포인트**(아주 조금만): **골드/앰버** (#C9A227, #B8860B) — “엔코어”, “커튼콜” 느낌.
  - 사용처: 로고 옆 작은 포인트, 주요 CTA 호버 보더, 푸터 상단 라인 등 **한두 군데만**.
- 효과: 색은 3개지만 골드는 면이 아니라 선/작은 요소만 써서 트렌디하고 과하지 않게.

### 추천 조합 (상담 기준)

- **색깔 바꿀까 고민**이라면:  
  **옵션 A(배경만 웜 화이트) + 옵션 B(메인만 딥 레드로)** 까지 적용해 보는 걸 추천.  
  Litera + #333 은 유지하고, 레드만 #B91C2E 정도로 바꾸면 “깔끔 + 무대”가 잘 맞습니다.
- **조금 더 “ENCORE”를 강조**하고 싶으면:  
  옵션 C처럼 골드/앰버를 **한두 요소에만** 넣는 걸 나중에 추가해도 됩니다.

## 4. 적용 구조 제안 (머지 후 CSS 정리 시)

1. **공통 theme 1개**
   - `static/css/theme.css` (또는 `common.css`)
   - `:root` 에 변수 정의:
     - `--encore-primary`, `--encore-secondary`, `--encore-text`, `--encore-bg`, `--encore-bg-warm`
     - `--encore-font-size-base` (18px), `--encore-max-width` (960px) 등
   - PC 메인 `html { font-size: ... }`, 모바일 `@media` 폰트/레이아웃

2. **페이지 로드 순서**
   - Bootstrap(Litera) → **theme.css** → 페이지별 CSS  
   - 페이지별에서는 `#E63946` / `#333` 하드코딩 대신 `var(--encore-primary)` 등 사용하면, 나중에 색만 바꿔도 일괄 반영 가능.

3. **PC / 모바일**
   - 공통: `.app-wrapper` 등에 `max-width: var(--encore-max-width); margin: 0 auto;`
   - 768px 이하에서만 `max-width: 100%` 또는 430px 등으로 제한.

이렇게 하면 “PC 메인 + 발표용으로 폰트/크기 키움 + Litera 유지 + 색만 살짝 무대감 있게” 가져갈 수 있습니다.

---

## 5. 요약

| 항목 | 제안 |
|------|------|
| **PC/모바일** | PC 메인, `max-width` 960px급, 모바일만 미디어쿼리로 제한 |
| **폰트** | 기본 18px(발표용), 모바일 16px |
| **테마** | Litera 유지, 색은 변수로 한곳에서 관리 |
| **색상** | 배경 웜 화이트(#FAF9F7) + 메인 딥 레드(#B91C2E) 검토, 서브 #333 유지. (선택) 골드 포인트 소량 |
| **느낌** | 깔끔·트렌디·과하지 않게, “무대를 잇다 ENCORE” 가 살짝 느껴지도록 |

머지 후 CSS 총정리할 때 이 순서(공통 theme → 변수 → PC 메인 폰트/레이아웃 → 색상 선택)로 적용하면 됩니다.

## 맛보기용 theme.css (롤백해 둔 상태)

- **파일**: `static/css/theme.css` — 웜 배경, 딥 레드, 18px, 960px wrapper 등 적용.
- **적용**: 사용하려면 적용할 페이지 `<head>`에 맨 뒤에 추가.
  ```html
  <link rel="stylesheet" th:href="@{/css/theme.css}" />
  ```
- **예**: 피드에 다시 쓰려면 `templates/feed/list.html`에서 `list.css` 다음에 위 한 줄 추가.
